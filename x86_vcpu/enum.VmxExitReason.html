Expand description

VMX basic exit reasons, as defined in the Intel Software Developerโs Manual (SDM) Vol. 3D, Appendix C.

This enum represents the various reasons why a VM exit might occur during the execution of a virtual machine in VMX (Virtual Machine Extensions) mode. Each variant corresponds to a specific exit reason that can be identified and handled by the hypervisor.
